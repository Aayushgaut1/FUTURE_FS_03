The Roasted Bean – Café Website

A responsive website for a local café, built using HTML, CSS, and JavaScript. Designed to showcase a small business online and ready to customize for real-world use.

Overview

This project provides a clean, modern website structure for a café, including:

Hero section

About section

Menu highlights

Opening hours & location

Contact section

Responsive navigation

The layout is mobile-friendly and suitable for small businesses needing a simple online presence.

Tech Stack

HTML5

CSS3 (Flexbox, Grid, responsive design)

Vanilla JavaScript

Google Fonts (DM Serif Display, Outfit)

Run Locally

Open index.html in a browser

OR

npx serve .
Customization

Before presenting to a real café owner, update:

Business name

About section content & images

Menu items

Hours & address

Contact details

You can also connect the contact form to services like Formspree or Netlify Forms.

Business Value

This website helps a café:

Build a professional online presence

Make information easily accessible

Improve mobile discoverability

Maintain a low-cost digital solution
